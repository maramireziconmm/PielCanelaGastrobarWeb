# Piel Canela Gastrobar — Sitio estático

Contenido listo para desplegar. Estructura principal:
- index.html (Inicio / Especialidades)
- raciones.html
- menus.html
- vinos.html
- horarios.html
- contacto.html
- assets/
  - css/style.css
  - js/includes.js
  - partials/header.html
  - partials/footer.html
  - images/  ← aquí debes subir todas las imágenes

Instrucciones rápidas:
1. Coloca todas las imágenes en `assets/images/` con los nombres indicados (lista abajo).
2. Comprueba que la estructura de carpetas es exacta.
3. Abre `index.html` en tu navegador o sube el contenido al repositorio.
4. Si quieres usar GitHub Pages: crea una rama `gh-pages` o usa `main` y en Settings → Pages selecciona la rama donde subas los archivos.

Notas:
- El header y footer se incluyen mediante `assets/js/includes.js`. Para pruebas locales con file:// puede haber limitaciones por CORS al usar fetch; si ocurre, sirve el sitio con un servidor local (p. ej. `python -m http.server`).
- Los nombres de imágenes deben coincidir exactamente con los usados en los HTML. Si cambias nombres, actualiza las rutas en los archivos.
- Puedo preparar una PR una vez subas este contenido al repo (si quieres que lo haga por ti).
